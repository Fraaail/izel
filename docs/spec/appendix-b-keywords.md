# Appendix B: Keyword Reference (Normative)

v1.0 keyword set is defined by the language overview lexical chapter and lexer token definitions.

## Categories

- Declaration keywords (`forge`, `shape`, `scroll`, `weave`, `ward`, etc.)
- Control keywords (`given`, `else`, `loop`, `while`, `each`, `branch`, etc.)
- Effect/memory-safety related keywords (`raw`, `bridge`, `zone`, `flow`, `tide`)

## Reservation Rules

Keywords are reserved unless explicitly documented as contextual.
Future editions MAY promote reserved words to active keywords but SHOULD provide migration notes.
